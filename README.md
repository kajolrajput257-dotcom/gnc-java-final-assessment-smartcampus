# SmartCampus Management System

This is a Java-based application developed for *GNC College* as part of the BTech 2nd Year Final Assessment.

## Features
- Student Management: Add and track student records.
- Course Management: Add courses with fee validation.
- Asynchronous Enrollment: Uses Multithreading to process enrollments in the background.
- Custom Exception: Handles errors like negative fees using a custom exception class.

## Technical Concepts Used
- OOPs: Classes, Objects, and Constructors.
- Collections: ArrayList for student/course lists.
- Exception Handling: Try-catch blocks and Custom Exceptions.
- Multithreading: Thread class for background tasks.

## How to Run
1. Open the folder in VS Code or Terminal.
2. Compile the project:
    javac SmartCampusSystem.java`
3. Run the application:
    java SmartCampusSystem`

## Sample Output
- Menu with 5 options.
- Enrollment processing message from background thread.
